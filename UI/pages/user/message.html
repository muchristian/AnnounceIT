EPIC mail

1. On Pivotal Tracker, create user stories to setup the User Interface (UI) elements: a. User sign-up page. b. User sign-in page. c. User can reset password. d. A page/pages where an admin can do the following: i. Create a group and add different users to the group. e. A page/pages where a user can do the following: i. Send email to individuals or groups. ii. View inbox. iii. Read an email message and any resulting thread. iv. View sent messages and retract a sent message. v. Save a message as draft and either send it later or delete it. 2. On Pivotal Tracker, create stories to capture any other tasks not captured above. A task can be feature, bug or chore for this challenge.
